HTML minify
=============================
Enables Smarty 'trimwhitespace' Outputfilter to minify HTML Code

Installation
------------

1.    Activate module in the OXID backend

----

After uploading the module, your directory should look like this:

```
├── index.php
├── bootstrap.php
├── config.inc.php
├── ...
└── modules
    └── ab_htmlmin
        ├── README.md
        ├── metadata.php
        └── models
```
